# Preferred sources for competitive-analysis research

Researcher subagents should bias searches toward these domains. Use `site:` queries when a source is high-signal.

## Recency rule (hard)

**Do not cite sources older than 5 years.** Measured against today's date at research time (e.g., in 2026, cut off everything dated before 2021-01-01).

- Applies to all factual claims in briefs and all sources rendered in the deck appendix.
- Exceptions — allowed only if flagged explicitly in-line as "(historical)":
  - Founding dates, accreditation dates, and other immutable institutional facts.
  - Wikipedia overview articles used only as navigation, never as the primary citation for a live claim.
- If a number is only available from a stale source (e.g. 2013 ad-spend figure), either drop the claim or label it in the brief as "stale — flag in deck" so the deck-architect can show the caveat on the slide (as done with the 2013 COLMAN ad-spend reference).
- When a user-facing claim depends on a stale source, prefer dropping the claim over smuggling it in.

## Search query patterns

**Hebrew (for Israeli brands or Israeli operations of global brands):**
- `"<brand>" 2025 site:calcalist.co.il`
- `"<brand>" השקה` (launch)
- `"<brand>" קמפיין` (campaign)
- `"<brand>" הכנסות` (revenue)
- `"<brand>" מנכ"ל` (CEO)
- `"<brand>" מתחרים` (competitors)

**Hebrew — marketing-specific (for marketing-researcher):**
- `"<brand>" פרסומת site:ice.co.il`
- `"<brand>" קמפיין 2025`
- `"<brand>" סוכנות פרסום` (advertising agency)
- `"<brand>" תקציב מדיה` (media budget)
- `"<brand>" ריברנד` / `ריבראנד` (rebrand)
- `"<brand>" פרסומת השבוע`
- `"<brand>" Yifat` (media monitoring citations)

**English:**
- `"<brand>" earnings 2025`
- `"<brand>" product launch`
- `"<brand>" marketing campaign`
- `"<brand>" leadership change`
- `"<brand>" Israel`
- `"<brand>" agency of record`
